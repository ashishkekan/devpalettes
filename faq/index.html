Frequently Asked Questions

Quick answers to common questions about our tools.

Is Devpalettes free to use?

Yes! Devpalettes is 100% free for personal and commercial use. You can generate unlimited palettes, export CSS code, and use our tools without any cost or subscription.

How do I export my color palette?

Simply click the "Export CSS" or "Copy" button on any palette card. You can copy individual colors or the entire CSS code block to use in your projects.

Can I use these colors for commercial projects?

Absolutely. All color combinations and palettes generated are free from copyright restrictions. You can use them in websites, apps, branding, and any other commercial design work.

What formats can I export?

We support CSS (CSS Variables), JSON, and Tailwind CSS config formats. You can also copy individual HEX, RGB, and HSL values from the color converter tool.

What types of gradients can I create?

Our gradient generator supports both linear and radial gradients with multiple color stops. You can adjust angles, add unlimited colors, and copy the CSS code instantly. Use presets for quick inspiration or create custom gradients from scratch.

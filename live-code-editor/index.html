Frequently Asked Questions

Is my code stored on any server?

No. All code stays in your browser's localStorage. Nothing is sent to any server.

Can I use this editor offline?

Once loaded, the editor works offline. Your code is auto-saved locally and persists across sessions.

How do I export my code as separate files?

Click the download icon and select "Export as ZIP". You'll get a ZIP with separate index.html, style.css, and script.js files.

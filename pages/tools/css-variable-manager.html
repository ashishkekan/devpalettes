What are CSS Variables?

CSS Variables (Custom Properties) allow you to store values for reuse in your stylesheet. They make it easy to maintain themes, switch between dark/light modes, and keep your code DRY (Don't Repeat Yourself).

:root { --main-color: #333; }
body { color: var(--main-color); }

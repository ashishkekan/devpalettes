Getting Started

Welcome to the Devpalettes Help Center, your central hub for guides, support, and tutorials covering every free color tool we offer. Whether you are building a brand identity, designing a user interface, or exploring color theory, our Help Center provides the documentation you need to get productive quickly. No account is required, and every utility runs entirely in your browser.

Start with our Color Palette Generator to build harmonious schemes, or jump into the gradient builder for smooth, multi-stop transitions. If you need to translate values between formats, the Color Converter handles HEX, RGB, and HSL instantly. For deeper reading, browse our blog where we publish in-depth articles on color theory and accessibility.

Using Devpalettes Tools

Each tool in our Help Center is free, requires no sign-up, and works on both desktop and mobile browsers. Below is a quick overview of the four core workflows available on the platform.

Color Palette Generator

Generate palettes from a single base color or explore curated combinations. Adjust hue, saturation, and lightness, then export as CSS variables, Tailwind config, or JSON. Lock individual swatches while you randomize the rest of the set. Visit the palette generator to start creating.

Gradient Generator

Build linear and radial gradients with unlimited color stops. Copy the generated CSS, adjust angles visually, and preview results under both light and dark themes. Try the gradient tool to design smooth color transitions.

Color Converter

Paste any HEX, RGB, HSL, or named CSS color to receive every equivalent representation instantly. The converter also surfaces the nearest accessible color name for quick reference.

Contrast Checker

Verify that foreground and background pairs meet WCAG AA and AAA thresholds. The contrast checker references the official W3C WCAG 2.1 Quick Reference so you can ship accessible interfaces with confidence. Learn more about vision differences on our color vision guide.

Troubleshooting

If a tool in our Help Center behaves unexpectedly, try these steps first:

  • Clear your browser cache and reload the page.
  • Disable extensions that modify stylesheets.
  • Update to a modern, supported browser version.
  • Review release notes on our blog for recent platform changes.

To report a bug, use our contact form and include your browser version, the tool URL, and a short description of the issue. Bug reports help us improve the platform for everyone.
